Checkout

  1. Checkout or download this repository locally.
  2. Go to chrome://extensions/ in your browser and turn on Developer mode in the top right.
  3. Press Load unpacked and select the src folder. Do not remove this folder, removing it will cause the extension to be uninstalled.
  4. Resume coding with closed issues being red.

Summary

Revert closed GitHub issues from purple back to red.

Description

Created to revert the GitHub issue color change that was forced onto users without any feature preview or user consent. Extension is compatible with Chromium based browsers and Firefox. Uses minimal JavaScript and mostly complex CSS selectors.
